Anthropic überholt OpenAI.
Drei Mal in 72 Stunden.

KW 22 / 2026 25.–31. Mai 5 Hersteller · 12 Meldungen

965 Milliarden Bewertung. Opus 4.8 schlägt GPT-5.5. Fast Mode jetzt drei Mal günstiger. Während OpenAI auffällig still bleibt und Meta in den Sicherheits-Layer flüchtet, schreibt Anthropic die Rangordnung der Branche um. Und kündigt mit Mythos das erste Modell an, das offen mit Cyber-Capabilities verkauft wird.

Anthropic
Release Claude Opus 4.8 — gleicher Preis, bessere Zahlen
Am 28. Mai released, direkter Nachfolger von Opus 4.7 zum gleichen API-Preis ($5 Input / $25 Output pro Mio. Tokens). SWE-Bench Verified: 88,6 % (4.7: 87,6 %). Terminal-Bench 2.1: 74,6 %. GPQA Diamond: 93,6 %. GDPval-AA: 1.890 Elo (4.7: 1.753) — aktuell führend im Markt. Vier operative Neuerungen: Parallel-Subagent Dynamic Workflows in Claude Code, Mid-task System Messages auf der Messages API, ein neuer Fast Mode und messbare Honesty-Verbesserungen im Alignment Assessment.
Pricing Fast Mode drei Mal günstiger — Opus wird operativ
Die unterschätzte Nachricht der Woche. Fast Mode kostet jetzt $10 Input / $50 Output pro Mio. Tokens, vorher $30 / $150 — drei Mal günstiger bei 2,5-facher Geschwindigkeit. Wer Opus-Agents wegen Token-Kosten verworfen hat, sollte ab dieser Woche neu rechnen. Effektive Cost-per-Task im Agent-Workflow liegt deutlich unter Opus-4.7-Niveau.
Funding 65 Mrd. Series H — Bewertung 965 Mrd. $
Geleitet von Altimeter, Dragoneer, Greenoaks und Sequoia, ko-geleitet von Capital Group, Coatue, D1, GIC, ICONIQ und XN. $15 Mrd. davon committet von Hyperscalern, davon allein $5 Mrd. von Amazon. Die Bewertung überholt OpenAIs $852 Mrd. aus der März-Runde um $113 Mrd. Die beiden Anthropic-Mitgründer werden jeweils mit ca. $8 Mrd. persönlichem Vermögen bewertet. Die zwei Jahre alte Erzählung vom ewigen Zweiten ist beendet.
Ankündigung Mythos — ein Modell mit Cyber-Capabilities
In der Pressemitteilung zur Funding-Runde angekündigt: Mythos-Modelle gehen "in den kommenden Wochen" an alle Kunden. Bemerkenswert ist die offene Positionierung — Anthropic vermarktet die Fähigkeit, Software-Schwachstellen zu finden und zu Angriffsketten zu verbinden, als Feature. Sicherheitsbedenken bei kritischer Infrastruktur (Stromnetz, Finanzsystem) werden im Funding-Statement adressiert. Die Wette: Pen-Testing-Anwendung überwiegt den Backlash. Diese Wette ist nicht ausgemacht.
xAI · Grok
Release Grok Build 0.1 in Public Beta — 29. Mai
xAIs schnellster Coding-Agent ist jetzt über die xAI API verfügbar. Bisher nur für SuperGrok-Heavy-Subscriber ($300/Monat) zugänglich. Damit greift xAI Anthropics Claude Code und OpenAIs Codex direkt im Pricing-relevanten Mid-Tier an.
Feature Custom Skills offiziell — 26. Mai
Persistente, wiederverwendbare Automation-Tasks. User können personalisierte Skills in Sekunden bauen und täglich deployen. Im Gegensatz zu Memory: explizit, parametrisierbar, teilbar.
Integration Grok Connectors + Imagine Quality Mode
Tiefe Integrationen in Grok Web für SharePoint, Outlook, OneDrive, Google Workspace, Notion, GitHub und Linear. Parallel ging der Quality Mode der Grok Imagine API live für Enterprise-Developer — mit höherem Realismus und besserer Text-Rendering.
Meta
Release Llama Security Stack — 26. Mai
Drei neue Tools auf einmal: Llama Guard 4 filtert jetzt multimodal — nicht mehr nur Text, sondern auch Bilder. LlamaFirewall agiert als Security Control Center für AI-Systeme und koordiniert mehrere Safety-Modelle. Prompt Guard 2 (86M) hebt die Jailbreak- und Prompt-Injection-Erkennung auf ein neues Niveau.
Strategie Sicherheits-Layer statt neues Foundation Model
Hintergrund: Meta hat das nächste Llama-Foundation-Modell laut Reports verschoben und investiert verstärkt in das Tooling rund um die bestehende Generation. Sicherheits-Stack ist die Antwort auf Enterprise-Anforderungen — und ein Versuch, ohne neue Modell-Generation relevant zu bleiben.
OpenAI
Stille Kein offizielles Release in KW 22
Mitte Mai tauchte GPT-5.6 kurz in Codex Backend-Logs auf und wurde dann auf 5.5 zurückgerollt. Konsistent mit interner Canary-Phase, kein Launch. Polymarket pricet die Wahrscheinlichkeit für ein Public Release bis 30. Juni mit 80–89 % ein. Sonst kein neues Produkt-Statement in einer Woche, in der Anthropic die Marktführerschaft übernimmt.
Retirements o3 und GPT-4.5 enden
Sunset-Daten in den ChatGPT Release Notes bestätigt: GPT-4.5 wird am 27. Juni 2026 aus ChatGPT entfernt (30-Tage-Sunset). o3 wird am 26. August 2026 entfernt (90-Tage-Sunset). Wer Workflows auf diese Modelle festgenagelt hat, sollte Migration jetzt einplanen.
Google · DeepMind
Rollout Gemini Spark Beta für US Ultra-Subscriber
Diese Woche für Google AI Ultra Subscriber ($100/Monat) in den USA freigeschaltet. Personal-AI-Agent auf Antigravity-Basis, läuft auf Gemini 3.5, agiert autonom mit Bestätigungspflicht vor größeren Aktionen. Sub-Agent-Erstellung und Zahlungs-Autorisierung mit Budget-Limits für den Sommer geplant. Gemini 3.5 Pro bleibt im Testing, Public Release im Juni angekündigt.

Diese Woche hat die KI-Rangordnung zum ersten Mal seit GPT-4 gewechselt — und nicht durch ein einzelnes Release, sondern durch drei innerhalb von 72 Stunden.

Anthropic ist jetzt das wertvollste KI-Startup, hat das stärkste Frontier-Modell und kündigt mit Mythos die nächste Generation an, bevor OpenAI auf das letzte Release antwortet. Für Controller heißt das: Vendor-Konzentration neu denken, Token-Budgets aktualisieren, Compliance für Cyber-Capabilities vorbereiten. Wer 2024/25 strategisch auf OpenAI gesetzt hat, sollte spätestens jetzt eine Anthropic-Alternative ko-pilotieren. Marktführer wechseln, Lock-in wird teurer.
